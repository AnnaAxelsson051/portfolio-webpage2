Law website

This is a law website I have built using Html, Css and Javascript with extensive information regarding swedish testament law. The user is able to navigate between different topics and click to read more about each subject. This site includes information regarding how a persons belongings is distributed after death, what requirements there are when it comes to writing a legally valid will, what the rules and principles are surrounding emergency wills, how to retract a will, what makes a will legally invalid and under what circumstances it is possible to loose the right to inherit from a will as well as much more. This website is not final and planned to later also include back end with a search function, a database with interesting precedents (cases) and more.

Hajabadollah Game

This is a fun and creative game I have built using Html Css and Javascript. Help Hajabadollah (a persian candy king) to beat Jannati (leader of the iranian regimen and opressor of the people in Iran) by collecting the most points. First player to hit 80 points wins. The game is played by spinning a wheel which makes different types of persian delicatesses randomly appear in the form of images, each generating different points. One image can also cause the player to loose all points.. Play two or play as Hajabadollah against the computer (Jannati). I would like to enhance this game in several ways, adding animations, sound effects etc.

Java Menu selector

This is a back end Java app I have built, where the user is faced with a number of options to select from a menu. The user is for instance able to play rock, paperbag, scissor against the computer. Another option allows the user to enter a row of numbers seperated by spaces (as a string) the app sorts it as an integer array and returns the second min and second max value. Another option uses arraylist to take an unlimited number of strings as input and on the users command returns them in reverse order. I expect this kind of app of mine to be more advanced and perfected in the future.
